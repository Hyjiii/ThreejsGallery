사용자가 점심, 저녁 메뉴를 좀 더 쉽게 고르기 위해 만든 앱으로서,
무작위로 음식을 추천하는 기능과 음식 종류와 맛에 맞게
단계적으로 정확하게 음식 메뉴를 고를 수 있게 안내하는 앱

* 개발 목적

항상 점심이나 저녁 메뉴를 쉽게 고르지 못해서 소비하는 시간이 아깝다고 생각하여
나에게 그때 당시에 가장 먹고 싶은 음식을 찾아주는 APP을 개발하게 되었다.
점심이나 저녁 메뉴를 고르는 것에 대해 이 APP을 통해 많은 사람이 시간을 절약할 수 있는 것이 목적이다.

* 사용법

1. 회원 가입을 통해 사용자 계정을 만든 후 로그인을 한다.
2. 사용자의 스마트폰의 GPS 기능을 활성화한다.
3. 사용자는 무작위로 음식을 추천받는다. 음식 사진들 중 취향에 맞는 음식사진이 있을 경우 클릭한다.
(없을 경우 직접 고르기 항목을 선택한다.)
4. 직접 고르기에서는 음식의 종류(한식, 중식, 일식, 양식)과 음식의 맛(단맛, 짠맛, 매운맛 등)에서
하나씩 선택 후 선택 완료를 누른다.
5. 해당 종류와 맛에 맞는 음식 사진들 중 사용자가 원하는 음식의 사진을 선택한다.
6. 해당 음식이 있는 주변 식당의 위치와 정보(평점, 다른 사용자의 평가) 등을
‘Google Maps’ 연동을 통해 보여준다.
